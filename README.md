# homebrew-sin

Homebrew tap for SIN-Code tools.

## Installation

```bash
brew tap OpenSIN-Code/sin
brew install sin-code
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `sin-code` | SIN-Code — verification-first coding agent (90+ subcommands, 52+ MCP tools) |

## Updating

Formulae are automatically updated by goreleaser on each release tag (`vX.Y.Z`).
